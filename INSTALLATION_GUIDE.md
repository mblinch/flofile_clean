# FloFile Beta - Installation Guide

## 🚀 Quick Start Installation

### Step 1: Download and Mount
1. Download the `FloFile_Beta_*.dmg` file
2. Double-click the DMG file to mount it
3. You'll see a window with the FloFile Beta app and an Applications folder

### Step 2: Install the App
1. Drag the **FloFile Beta** app to the **Applications** folder
2. Wait for the copy to complete
3. Eject the DMG (drag to trash or right-click → Eject)

### Step 3: First Launch - Security Settings
**This is the important part!** When you first try to open FloFile Beta, macOS will block it.

#### Option A: System Preferences Method (Recommended)
1. Go to **System Preferences** (or **System Settings** on newer macOS)
2. Click **Security & Privacy** (or **Privacy & Security**)
3. Click the **General** tab
4. You'll see a message: *"FloFile Beta was blocked from opening because it is not from an identified developer"*
5. Click **"Open Anyway"** or **"Allow"**
6. Click **"Open"** in the confirmation dialog

#### Option B: Right-Click Method
1. In Finder, go to **Applications**
2. Right-click (or Control+click) on **FloFile Beta**
3. Select **"Open"** from the context menu
4. Click **"Open"** in the security dialog that appears

### Step 4: Verify Installation
- The app should now open normally
- You'll see the blue rocket icon in your Applications folder
- Future launches won't require this security bypass

## 🔧 Troubleshooting

### If the app still won't open:
1. Make sure you're using one of the methods above (not just double-clicking)
2. Check that you're running macOS 10.15 (Catalina) or later
3. Try restarting your Mac and repeating the process

### If you get a different error:
- The app might need additional permissions for file access
- When prompted, allow access to Documents, Desktop, or Downloads folders
- This is normal for apps that work with your files

## 📝 Why This Happens

This security feature is called **Gatekeeper** and it's designed to protect your Mac from malicious software. Since FloFile Beta is distributed as a DMG (not through the App Store), macOS treats it as "unidentified" software.

**This is completely normal and safe** - it's just macOS being extra cautious about apps that aren't from the App Store.

## 🎯 What to Expect

Once installed, FloFile Beta will:
- Work normally like any other app
- Have access to your image folders (when you grant permission)
- Save captions and metadata to your photos
- Connect to baseball APIs for team and player data

## 🆘 Need Help?

If you're still having issues:
1. Make sure you downloaded the latest version
2. Try the right-click method if System Preferences doesn't work
3. Check that your macOS version is up to date
